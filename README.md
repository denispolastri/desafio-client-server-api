# desafio-client-server-api
Desafio para aplicar conhecimentos de HTTP, Contextos, Banco de Dados e Manipulação de Arquivos em Go.

### Instruções do projeto
```
|_client
|   client.go
|_server
|   server.go
|.gitignore
| dolar.db
| go.mod
| go.sum
| main.go
```

O arquivo main.go deve ser executado na raiz do projeto, na pasta desafio-client-server-api.
